Internet Explorer 5.5 with Service Pack 3 Support Extended

Updated Clarification on the Internet Explorer 5.5 with Service Pack 3 Support Extended Announcement

We are pleased to announce a clarification in our extended security update support for Internet Explore 5.5 with Service Pack 3 for critical security issues. As part of this extended support, information will be provided on whether critical issues affect these operating systems. We will provide security updates on affected operating systems and the severity rating is critical.

Key Dates:

  • Payment incident support for Internet Explorer 5.5 Service Pack 3 regardless of operating system is available indefinitely.
  • Critical security updates will be provided on this site indefinitely.
  • Customers can request non-critical security fixes for Internet Explorer 5.5 with Service Pack 3 and the latest version of its components indefinitely through typical assisted support channels.
  • Internet Explorer 5.5 Service Pack 3 downloads for existing security issues will continue to be available through regular assisted support channels at no charge indefinitely.

Details:

Support for Internet Explorer 5.5 Service Pack 2 was scheduled to end on July 11, 2006. Ongoing evaluation of the support lifecycle policy revealed, however, that customers in smaller and emerging markets needed more time. to update your product. Therefore, Internet Explorer 5.5 Service Pack 3 will continue to be supported indefinitely.
