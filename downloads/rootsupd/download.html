Root Certificates Update




Read This First

This article describes an update that enables urgent updates for the Windows Root Certificate Program in Microsoft Windows 95, Windows 98, Windows 98 Second Edition, Windows Millenium Edition, Windows NT 4.0, Windows 2000, Windows XP and Windows Server Editions. Before you apply this update, see more about this update and check out the prerequisites in this article.

The Windows Root Certificate Program enables trusted root certificates to be distributed automatically in Windows. Usually, a client computer polls root certificate updates one time a week. After you apply this update, the client computer can receive urgent root certificate updates within 24 hours.
